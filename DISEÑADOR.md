# Agente DISEÑADOR - Guía de Uso

## 📍 Ubicación
```
~/.openclaw/agents/disenador/AGENT.md
```

## 🎯 Propósito
Este agente es un experto en UX/UI que puede:
- Revisar landing pages
- Sugerir mejoras de diseño
- Reemplazar iconos genéricos con SVG profesionales
- Aplicar principios de conversión

## 🚀 Cómo activarlo

Por ahora, para ejecutar el agente necesitas:

1. **Agregar a allowlist** en la configuración de OpenClaw
2. **O ejecutarlo manualmente** leyendo las instrucciones del AGENT.md

## 💡 Ejemplo de uso

Cuando necesites diseñar algo nuevo:
1. Describí tu idea
2. Yo aplico los principios del agente diseñador
3. Iteramos hasta que quede perfecto

## 🎨 Recursos que usa

- **Iconos:** Lucide Icons (https://lucide.dev)
- **Fuentes:** Google Fonts (Outfit, Space Grotesk, etc.)
- **CSS:** Tailwind CSS
